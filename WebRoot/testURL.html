To learn more, go to the RIMapper website... For the SVG maps you will need an SVG-capable browser:
  • Native SVG renderers: FireFox 1.5+, Opera 9+, Safari 1.3+ (or Chrome and other browsers based on WebKit), IE9+
  • or others using external libs: a Javascript library (eg. SVGweb or Rafael) or the Adobe SVG viewer (IE plugin).
Known issues and limitations:
  • Animation GUI TimeSlider only works in Opera and FireFox (basic animation works also in Safari and other WebKit based browsers).
  • Output formats other than SVG (png,jpeg) not possible in combination with GetGUI=true.
  • Extension to ISO 8601 [:1988(E)] not implemented: Only contiguous TIME periods (datetime_begin/datetime_end) for now.
  • External (cascaded) WMS layers are loaded in the background, without checking for succes or progress report
